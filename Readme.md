# 🌱 Pequeno Grão

Uma plataforma web desenvolvida como **Trabalho de Conclusão de Curso (TCC)** com o objetivo de oferecer informações, dicas e apoio para pais e mães de primeira viagem.

O projeto reúne conteúdos educativos, orientações sobre cuidados com o bebê, alimentação, amamentação, psicologia, curiosidades e um espaço de brechó infantil, proporcionando uma experiência completa para as famílias.

Este projeto foi desenvolvido como Trabalho de Conclusão de Curso com os objetivos de:

* Desenvolver uma interface intuitiva e acessível.
* Trabalhar em equipe utilizando Git e GitHub.


